## Clone Template
```sh
npx degit atqq/vite-vue3-js-template#main my-project
```
## Usage
### install dep
```sh
yarn install
```

### Dev
```sh
yarn dev
```

### Build
```sh
yarn build
```

### Preview Build
```sh
yarn serve
```
